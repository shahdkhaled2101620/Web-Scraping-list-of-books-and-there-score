# Goodreads Top 100 Books Web Scraper

This project is a Python web scraper that extracts the **top 100 highest-rated books on Goodreads** with at least 10,000 ratings.  
It collects the **book title, author, and average score**, and saves the data into Excel files. You can also merge multiple Excel files and clean them automatically.

## Features

- Scrapes Goodreads list page for books
- Collects book **Title**, **Author**, and **Score**
- Saves data into Excel
- Merge multiple Excel files into one
- Removes unnecessary columns like `Unnamed: 0` automatically

## Requirements

- Python 3.x
- Packages:
  - `requests`
  - `beautifulsoup4`
  - `pandas`
